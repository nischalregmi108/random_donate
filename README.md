This is a very basic experiment that demonstrates how ABM can answer some foundational problems of inequality studies. Suppose that there is a town where there are N households. Assume that these households exist for several hundred years; in a world where there is no population increase, this assumption is not absurd! Initially, all the households are equally wealthy – they have W units of wealth each. 

The town dwellers are very philanthropic. Each year, in a round-robin fashion, a household decides to donate 10% of its wealth to another randomly selected household in the same town. What will happen after 100 years? 200 years? Will all the households in the town remain equally wealthy?

The simulation tool will allow the user to play with the model parameters and observe the ouput.
